# GB Power Market Working Papers

This repository contains public working papers and research notes on the Great Britain power market.

## Working Paper 1

**Regime-Based Extreme Price Formation in the GB Power Market: Evidence from Price Bands, Fuel Mix, Interconnectors and REMIT Events, 2023–2025**

This working paper analyses extreme price formation in the GB power market using half-hourly system price, fuel mix, imbalance, interconnector and REMIT context data.

The main argument is that extreme prices are not usually caused by one single factor. Instead, they emerge from wider market regimes involving wind output, gas generation, imbalance direction, interconnector flows, timing and asset-availability context.
